# WeldWork.net — Next.js App

Это новая версия приложения WeldWork.net, написанная на Next.js 15.

## Стек технологий
- **Фреймворк:** Next.js (App Router)
- **Язык:** TypeScript
- **Стили:** Tailwind CSS, shadcn/ui
- **База данных/Бэкенд:** Supabase

## Запуск проекта

1. Перейдите в папку проекта:
   `cd app-next`
2. Установите зависимости:
   `npm install`
3. Скопируйте `.env.example` в `.env.local` и укажите ключи для Supabase:
   `cp .env.example .env.local`
4. Запустите сервер для разработки:
   `npm run dev`

Проект будет доступен по адресу: http://localhost:3000
